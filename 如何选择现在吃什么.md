# 如何决策吃什么

如何决策吃什么也是我做菜之前一大难题。所以只能用数学描述一下了。

## 计算方法

### 计算荤菜和素菜数量

* 总菜数量 = 人数 + 1
* 荤菜数量 + 素菜数量 = 总菜数量
* 荤菜数量 - 素菜数量 <= 1
* 荤菜数量 - 素菜数量 >= 0

* 上述方程组中，荤菜数量、素菜数量、总菜数量皆为整数。

求解上述方程组，得到荤菜数量和素菜数量即可。
